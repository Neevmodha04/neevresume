Online Food Ordering System

PHP MySQL JavaScript CSS HTML

A comprehensive web-based food ordering system designed to streamline the process of ordering food online. This project implements a full stack solution with user authentication, order management, and an admin dashboard.

Key Features:

  • User registration and authentication system
  • Menu browsing with search and filter functionality
  • Shopping cart management
  • Order tracking and history
  • Admin panel for restaurant owners to manage menu items and orders
  • Responsive design for mobile and desktop devices
  • Secure payment processing simulation

Technical Details:

The backend is built with PHP and MySQL, handling database operations and business logic. The frontend uses HTML, CSS, and JavaScript to create an interactive user interface. The system follows MVC architecture for better code organization and maintainability.

Library Management System

Java Swing MySQL JDBC

A desktop application built with Java Swing to manage library operations including book inventory, member management, book issuing and returns, and fine calculations for late returns.

Key Features:

  • Book management (add, update, delete, search)
  • Member registration and management
  • Book issuing and return processing
  • Automatic fine calculation for overdue books
  • Reporting system for books, members, and transactions
  • User authentication for librarians and administrators
  • Data export functionality

Technical Details:

The application is developed using Java Swing for the GUI and JDBC for database connectivity with MySQL. It follows object-oriented principles with clear separation of data access, business logic, and presentation layers.

Interactive Quiz Game

JavaScript HTML5 CSS3 Web Audio API

An engaging web-based quiz application featuring timed questions, multiple difficulty levels, sound effects, and animations to enhance the user experience.

Key Features:

  • Multiple categories of quiz questions
  • Three difficulty levels: Easy, Medium, and Hard
  • Timed questions with visual countdown
  • Score tracking and high score leaderboard
  • Sound effects and animations for correct/incorrect answers
  • Responsive design for all device sizes
  • Progress tracking across quiz sessions

Technical Details:

The quiz game is built using vanilla JavaScript for the logic, HTML5 for structure, and CSS3 for styling and animations. It utilizes the Web Audio API for sound effects and localStorage for persisting user data and high scores.

Resume Website

HTML5 CSS3 JavaScript Responsive Design AOS Library

This personal resume website showcases my skills, experience, and projects in an interactive and visually appealing manner. It serves as both a portfolio and a practical demonstration of my web development abilities.

Key Features:

  • Responsive design that works on mobile, tablet, and desktop devices
  • Animated elements using AOS (Animate On Scroll) library

  • Project showcase with detailed modal views

Technical Details:
